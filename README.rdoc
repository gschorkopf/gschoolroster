== Welcome To your gRoster

gRoster is a simple [rails](http://rails.org) app using the [twitter bootstrap cdn](http://bootstrapcdn.com).


== Getting Started

1. Clone the Repo

2. Change directory to <tt>gRoster</tt> and `bundle install`

3. Run `rake db:migrate' and then start your server with 'rails s'

4. point your browser to [http://0.0.0.0:3000](http://0.0.0.0:3000) and
